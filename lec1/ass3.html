Observations

  • Div is a block-level element.
  • Each div starts on a new line and occupies the full available width.
  • Span is an inline element.
  • Span elements stay on the same line unless there is no space available.
  • Div is generally used for page layout and grouping larger sections.
  • Span is used for styling or grouping small portions of text.
